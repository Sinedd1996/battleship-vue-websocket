# battleship game
Игра морской бой

# Локальный запуск проекта
node.js - например 20.15.0

1) Устанавливаем зависимости и запускаем сервер
./server - "yarn" либо "npm i" > в терминале вводим "node index.js"

2) Устанавливаем зависимости и запускаем клиент
./vue-app - "yarn" либо "npm i" > в терминале вводим "yarn dev" либо "npm run dev"

после успешного выполнения пунктов 1-2 открываем url http://localhost:5173/

# Запуск в Docker
в корне проекта в терминале вводим "docker-compose up"
открываем url http://localhost/
